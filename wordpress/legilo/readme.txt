Adds the free Legilo reading-aid widget to your website: contrast, font size, reading mask, read-aloud and more, in 17 languages.

== Description ==

Legilo is a free reading-aid widget: one button that lets your visitors adjust
contrast, font size, spacing, fonts (including a dyslexia font), a reading mask,
read-aloud and more. 17 languages with automatic detection, no tracking, no
cookies, no account.

Honest positioning: Legilo is a reading aid. It does NOT make your website
conform to WCAG, EN 301 549 or national accessibility laws - conformance is
created in your website's source code, not by a widget.

== Installation ==

1. Install and activate the plugin.
2. Optional: configure the widget at https://legilo.eu (position, colors,
   language, features) and paste the embed code under Settings, Legilo.
   Without configuration the default widget is used.

== Frequently Asked Questions ==

= Does this make my site accessible? =
No. Legilo is a comfort extra for your visitors, not a compliance tool.

= Does it load anything from third parties? =
Only the Legilo script itself (from legilo.eu or your own server if you
self-host the file). The widget sets no cookies and sends no data anywhere.
